 
Для запуска приложения на комьютере нужно 

1. Скопировать папку с приложением на комп

2. установить на компе Node.js (глобально)

3. Через консоль войти в папку и ввести команду npm i для доустановки на комп пакетов
используемых в приложении

4. запустить в корневой папки приложения через кансоль команду npm run serve

5. перейти в браузере по адресу http://localhost:8080/
